# 视频特效助手 - Mac版

## 使用方法

1. 下载 DMG 安装包
2. 双击安装，拖入 Applications
3. 打开软件，填写 API Key（左侧设置栏）
4. 拖入视频 → AI分析 → 预览 → 导出

## 支持的AI服务

## 系统要求

- macOS 12+
- 需要联网（调用云端AI）

## 从源码运行

```bash
npm install
npm start
```

## 打包

```bash
bash 一键打包.sh
```
