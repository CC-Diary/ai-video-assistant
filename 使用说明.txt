# 视频特效助手

## 使用方法

1. 双击 `一键打包.bat`，等待安装完成
2. 打开 `dist/` 目录，运行安装包
3. 打开软件，填写 API Key（左侧设置栏）
4. 拖入视频 → AI分析 → 预览 → 导出

- 需要联网（调用云端AI）
