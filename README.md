# AI视频加特效（联网版本）

AI自动识别关键词生成特效的桌面视频应用。

## 功能

- 🎙️ **双模式**：口播加特效 / 配字幕特效
- 🤖 **多AI模型**：DeepSeek（推荐）、通义千问
- 🎬 **6种特效**：标签、大数字、对比卡片、柱状图、金句、CTA
- 🗣️ **云端转录**：通义千问语音识别
- 🎨 **GSAP动画**：流畅的入场/退场动画效果
- 📦 **跨平台**：Windows + macOS

## 技术栈

- Electron 33
- Node.js + 原生 HTML/CSS/JS
- HyperFrames (HTML-to-Video)
- GSAP 动画引擎
- 通义千问 DashScope（语音转文字）
- DeepSeek / 通义千问（AI分析）
- ffmpeg 视频处理

## 安装使用

1. 下载 DMG 安装包
2. 双击安装，拖入 Applications
3. 打开软件，填写 API Key（左侧设置栏）
4. 拖入视频 → AI分析 → 预览 → 导出
